# Python Poetry

This project bootstraps a python package `mypackage`

## Poetry cheat sheet

- Update dependencies
  ```shell
  $ poetry update
  ```
- Build
  ```shell
  $ poetry build
  ```
- run project or command
    ```shell
    # Run using Poetry
    $ poetry run somecommand
    
    # Run without Poetry
    $ python -m mypackage 
    ```
- Publish on PyPI
    ```shell
    $  poetry publish
    ```
